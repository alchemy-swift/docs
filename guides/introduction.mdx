Welcome to Alchemy, a **modern, batteries included web framework for Swift**. You can use it to build a production ready backend for your next mobile app, cloud project or website.

```swift App.swift
@main
struct App: Application {
    func boot() {
        get("/hello") { req in
            "Hello, \(req["name"])!"
        }
    }
}
```
